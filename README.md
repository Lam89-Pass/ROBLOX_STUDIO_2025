# DOKUMENTASI ROBLOX STUDIO - GAME PROGRAMMING

Repositori ini berfungsi sebagai pusat dokumentasi dan penyimpanan source code untuk berbagai proyek Game Programming menggunakan Roblox Studio dan bahasa pemrograman Lua.

Teknologi yang Digunakan:
  - Game Enggine : Roblox Studio
  - Bahasa Pemrograman : Lua
  - Tools lain : VScode, Rojo, Git, dan AutoScale Lite.

Berikut adalah beberapa fitur utama yang didokumentasikan dalam repositori ini:
  - Music Playlist
  - Emote & Dance
  - Report
  - Shop
  - Hidden UI & Title
  - KDS Admin
